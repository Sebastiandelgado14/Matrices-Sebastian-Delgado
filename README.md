# Tarea 4 de operaciones con matrices organizadas en módulos

El programa permite realizar diferentes operaciones entre matrices mediante un menú interactivo.

## Funcionalidades del programa realizado 

El programa realiza las siguientes operaciones entre matrices:

1. Suma de matrices
2. Multiplicación de matrices
3. Producto de Hadamard
4. Producto de Kronecker

Además, el programa:

Valida errores en la entrada de datos

Detecta caracteres inválidos

Permite ingresar matrices de diferentes tamaños

Está organizado en módulos independientes


# Sobre la estructura del proyecto

El proyecto contiene los siguientes archivos:

- `main.py`
- `entrada.py`
- `menu.py`
- `operaciones_matrices.py`
